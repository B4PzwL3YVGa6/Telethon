ExportInviteRequest

Both users and bots may be able to use this method. See code examples.

---functions---
channels.exportInvite#c7560885 channel:InputChannel = ExportedChatInvite

Returns

ExportedChatInvite

This type can be an instance of either:

ChatInviteEmptyChatInviteExported

Parameters

channelInputChannelAnything entity-like will work if the library can find its Input version (e.g., usernames, Peer, User or Channel objects, etc.).

Known RPC errors

This request can cause 3 known errors:

ChannelInvalidErrorInvalid channel object. Make sure to pass the right types, for instance making sure that the request is designed for channels or otherwise look for a different one more suited.
ChatAdminRequiredErrorChat admin privileges are required to do that in the specified chat (for example, to send a message in a channel which is not yours).
InviteHashExpiredErrorThe chat the user tried to join has expired and is not valid anymore.

You can import these from telethon.errors.

Example

from telethon.sync import TelegramClient
from telethon import functions, types

with TelegramClient(name, api_id, api_hash) as client:
    result = client(functions.channels.ExportInviteRequest(
        channel='username'
    ))
    print(result.stringify())
